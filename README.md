# Lens Play


Lens Play is a video library of free programming tutorials.

# Tech Stack

- React JS
- React Router
- Mockbee (Backend)

## Features Present

- Authentication
  - Login
  - Logout
  - Signup
- Video Listing
- Filter Products by Category
- Sort videos by date
- Playlist Management
- Likes / Dislikes
- History
- Watch later
- Search by video title
- Single Video Page
- Loading & Alerts
